# Шаблон Feature Slice

При создании нового модуля в `src/features/` используй эту структуру:

```
src/features/[name]/
├── index.ts          # Единственный публичный вход
├── [name].service.ts # Бизнес-логика
├── [name].types.ts   # Типы и контракты
├── [name].test.ts    # Тесты рядом с кодом
└── [name].data.ts    # Конфигурации/таблицы (если нужно)
```

## Правила
- `index.ts` — единственный файл, через который импортируют другие модули
- Файл < 250 строк
- Тесты рядом с кодом
- Данные отделены от логики (data.ts vs service.ts)
