# Implement Feature

Реализуй фичу по утверждённому плану.

## Порядок действий:
1. Прочитай план (если есть plan.md или задачу из контекста)
2. Прочитай docs/BRD.md — найди релевантные RF и UC
3. Прочитай docs/DATA_DESIGN.md — найди релевантные таблицы
4. Прочитай docs/ARCHITECTURE.md — пойми где в пайплайне находится фича
5. Посмотри _reference/README.md — есть ли эталон для этого паттерна
6. Посмотри templates/feature/ — используй как каркас
7. Реализуй фазу за фазой
8. После каждого файла: typecheck + lint + тесты
9. Проверь что файл < 250 строк
10. Проверь что свопы только внутри convenience_tier + use_context
