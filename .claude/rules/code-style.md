# Code Style

- Файл < 250 строк. Больше — разбивай.
- Модуль экспортирует < 20 публичных функций.
- Нет `any` — используй `unknown` + type guards.
- Нет мутаций — возвращай новые объекты.
- Именование: camelCase для переменных/функций, PascalCase для типов/классов.
- Импорт только через `index.ts`. Нет прямых импортов из внутренних файлов модуля.
- Одна функция — одна ответственность.
