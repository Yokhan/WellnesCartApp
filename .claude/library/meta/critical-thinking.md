# Critical Thinking — Cognitive Biases & Reasoning Checklist

> Quick reference for decision-making. Full details: `.claude/skills/domain-science-review/SKILL.md`

## Top 10 Biases to Check

1. **Confirmation Bias** — Seeking evidence that confirms existing beliefs. FIX: consider-the-opposite; "what would change my mind?"
2. **Anchoring** — Over-relying on first info received. FIX: generate own estimate BEFORE seeing anchor.
3. **Availability Heuristic** — Judging by ease of recall, not frequency. FIX: use actual statistics.
4. **Sunk Cost Fallacy** — Continuing because of past spending. FIX: "If starting fresh, would I choose this?"
5. **Planning Fallacy** — Underestimating time/cost. FIX: reference class forecasting (how long did similar tasks ACTUALLY take?).
6. **Survivorship Bias** — Learning only from successes. FIX: study failures equally.
7. **Dunning-Kruger** — Least competent overestimate, experts underestimate. FIX: calibration, external feedback.
8. **Framing Effects** — Different conclusions from same data presented differently. FIX: reframe problem multiple ways.
9. **Status Quo Bias** — Preferring current state regardless of evidence. FIX: evaluate from neutral starting point.
10. **Groupthink** — Conformity suppressing dissent. FIX: devil's advocate, anonymous input.

## Evidence Hierarchy
Systematic Review > RCT > Cohort > Case-Control > Case Series > Expert Opinion > Anecdote

## Quick Checks Before Any Decision
- [ ] Am I seeking disconfirming evidence?
- [ ] What are the base rates?
- [ ] Is this one study or replicated finding?
- [ ] What would change my mind?
- [ ] Am I conflating correlation with causation?
- [ ] What's the effect SIZE, not just significance?
